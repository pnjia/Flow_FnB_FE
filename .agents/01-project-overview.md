# Project Overview

- **Project Name:** Mangkasir F&B (MVP Prototype).
- **Context:** Transitioning a basic retail POS into a complex F&B POS.
- **Tech Stack:** Next.js (App Router), Tailwind CSS, shadcn/ui components, Lucide React icons.
- **STRICT CONSTRAINT:** NO BACKEND. The entire system MUST rely on a Global State Management tool (Zustand or React Context) to simulate a database and real-time interactions across different routes.
- **Target Devices:** Mobile-first for QR Menu, Tablet/Desktop for POS and KDS.
