# UI/UX Rules

- **Component Library:** STRICTLY use `shadcn/ui` for interactive elements (Cards, Buttons, Badges, Dialogs, Checkboxes, RadioGroups, Switches).
- **Table Status Colors (Mandatory Mapping):**
  - "empty" -> Green / Default
  - "new_order" -> Orange / Yellow (Blinking indicator)
  - "cooking" -> Red
  - "ready_deliver" -> Blue (Blinking indicator)
  - "eating" -> Gray / Neutral
- **Design Philosophy:** Focus on functional flow, role-based interfaces (Customer vs Cashier vs Chef), and clear state visibility rather than pixel-perfect aesthetic details.
